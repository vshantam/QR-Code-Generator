Basic requirements are :
	1:->python installed (version python>=2 or python>=3)
	2:->required library:i>png
			    :ii>pyqrcode
			    :iii>qrcode
			    :iv>zbar
			    :v>qrtools
if not installed please installed using pip or easy_install commands.
'''example:-pip install package_name
	   #in case of png: pip install pypng
	   #in case of zbar :apt-get install libzbar-dev'''
or run the following command:
	sudo apt-get update && apt-get dist-upgrade 
	pip install -r spec.txt && apt-get install libzbar-dev
